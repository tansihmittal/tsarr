## How to Add More Posts

Simply create a new `.mdx` file in `content/blog/`:

```mdx
---
title: "Your Post Title"
excerpt: "Short description"
author: "Tanish Mittal"
publishedAt: "2025-02-01"
updatedAt: "2025-02-01"
category: "Tutorials"
tags: ["tag1", "tag2"]
featured: false
metaDescription: "SEO description"
keywords: ["keyword1", "keyword2"]
---

Your markdown content here...
```

The post will automatically:
- Appear on `/blog`
- Be included in sitemap
- Generate a static page at `/blog/your-slug`
- Include proper SEO meta tags
